Решение нелинейного уравнения методом деления отрезка пополам

Введение

Данный материал основан на этой статье.

Пусть имеется некоторая функция f(x), непрерывная на заданном интервале [x1, x2], причем на границах интервала функция имеет разные знаки. Это может быть выражено условием f(x1)*f(x2) < 0.

Требуется решить нелинейное уравнение f(x)=0, то есть найти такое x, при котором график функции пересекается с осью абсцисс, как показано на картинке. Красная точка соответствует искомому решению:

Последовательный ряд фигурных скобок внизу картинки обозначает интервалы аргумента (показаны только для первых четырех итераций). На каждой итерации интервал сужается вдвое. Процесс повторяется до тех пор, пока ширина интервала не окажется меньше заданной точности по x, либо модуль функции в очередном узле не окажется меньше заданной точности по у.

Описание алгоритма

Пусть заданы:

  • xn — начало отрезка по х;
  • xk — конец отрезка по х;
  • xi — середина отрезка по х;
  • eps_x — требуемая точность вычислений по x (ширина интервала);
  • eps_y — требуемая точность вычислений по y (модуль функции)

Тогда алгоритм решения нелинейного уравнения методом бисекции можно записать следующим образом:

  1. Ввод исходных величин xn, xk, eps_x, eps_y
  2. Если abs(f(xn)) < eps_y, то Вывод "xn — корень уравнения".
  3. Если abs(f(xk)) < eps_y, то Вывод "xk — корень уравнения".
  4. Если f(xn) * f(xk) > 0, то Вывод "Неверные исходные данные: на границах интервала функция имеет один и тот же знак".
  5. До тех пор, пока abs(xn - xk) > epsx, повторять действия:
    • xi = (xn + xk)/2
    • Если abs(f(xi)) < eps_y, выйти из цикла.
    • Если f(xi) * f(xn) < 0 , то
          xk = xi
      иначе
          xn = xi
    Вывод: "xi — корень уравнения".
